## Public Positioning

Core thesis:

> IDT is a candidate executable language for interpreting the physical structure
> of the universe, designed for both human researchers and AI scientific agents.
> Its goal is not to replace physics by assertion, but to provide a disciplined
> map from observations to readouts, from hypotheses to bounded bridges, and
> from assumptions to explicit anchors and gates. If matured, IDT could become
> a shared world-interpretation layer: a way for humans and AI systems to
> organize evidence, test explanations, expose gaps, and progressively deepen
> structural understanding of the physical world.

Inherited Distinguishability Protolanguage is an executable reconstruction
programme for testing whether quantum, relativistic, and weak-gravity readouts
can be constrained from a common primitive language of inherited
distinguishability.

The current value is not a finished replacement for QM or GR. The value is a
disciplined research artifact: explicit claim boundaries, no-refit gates,
calibrated anchor protocols, real-data fixtures, and a verifier that can turn
candidate bridges into auditable checks. If the programme succeeds, it could
identify a deeper shared structure behind multiple physical readouts. If it
fails, it should still produce useful exclusions and reusable verification
machinery.

Operationally, this means candidate explanations can be checked rather than
only narrated. In that role, the verifier is not just a guardrail; it is a tool
for helping humans and AI systems organize, test, and extend their world
models.

## Versioning Rule

No versioned copies of the whole theory should be created for normal development.

Each accepted version is:

1. edit the canonical module files;
2. commit;
3. annotate git tag.

Version numbers follow semantic versioning: `MAJOR.MINOR.PATCH`.
Annotated tags use the same version with a leading `v`, for example `v6.0.0`.
`MAJOR` is for incompatible canonical baselines, `MINOR` for compatible section
clusters or verifier capabilities, and `PATCH` for corrections that do not
change the accepted theory surface.

Historical full-file versions remain available through git tags.
