## 65. Step Clock Readout Rule

This section defines how the transfer-step time:

$$
\Delta\tau_{\mathrm{step}}
$$

can be obtained from clock readout without using gravity.

Status:

`step_clock_readout_rule_initialized`

### 65.1. Step count inside a clock interval

Let:

$$
N_{\mathrm{step}}(C;\Delta\tau_C)
$$

be the number of local transfer steps in a calibrated clock interval:

$$
\Delta\tau_C.
$$

Then:

$$
\Delta\tau_{\mathrm{step}}^{(C)}
=
\frac{\Delta\tau_C}
{N_{\mathrm{step}}(C;\Delta\tau_C)}.
$$

Clock universality requires:

$$
\Delta\tau_{\mathrm{step}}^{(C)}
=
\Delta\tau_{\mathrm{step}}^{(C')}
\left[
1+O(\epsilon_{\mathrm{step}})
\right].
$$

### 65.2. Relation to radar half-time

For a nearest-neighbour radar loop:

$$
\tau_{1/2}
=
\frac{\ell_{0,*}}{c_I}
=
\frac{1}{\omega_{\ell,I}}.
$$

Define:

$$
N_{1/2}
=
\frac{\tau_{1/2}}
{\Delta\tau_{\mathrm{step}}}.
$$

Then:

$$
\Delta\tau_{\mathrm{step}}
=
\frac{1}
{N_{1/2}\omega_{\ell,I}}.
$$

The step phase invariant is:

$$
\zeta_{\mathrm{step}}
=
\Delta\tau_{\mathrm{step}}\omega_{\ell,I}
=
\frac{1}{N_{1/2}}.
$$

`half_radar_step_count_relation`

### 65.3. Compatibility with cycle phase

Cycle grammar gives:

$$
\theta_*
=
\frac{2\pi m_*}{N_{\mathrm{cyc}}}.
$$

The first strain-coupled frequency is:

$$
\omega_{\ell,I}
=
\frac{\theta_*}
{\Delta\tau_{\mathrm{step}}}.
$$

Therefore:

$$
\theta_*
=
\Delta\tau_{\mathrm{step}}\omega_{\ell,I}
=
\zeta_{\mathrm{step}}
=
\frac{1}{N_{1/2}}.
$$

Closure requires:

$$
\boxed{
\frac{2\pi m_*}{N_{\mathrm{cyc}}}
=
\frac{1}{N_{1/2}}
}
$$

This is a strict arithmetic consistency condition between cycle grammar and clock-radar step count.

### 65.4. Consequence

If:

$$
N_{\mathrm{cyc}},
\qquad
m_*,
\qquad
N_{1/2}
$$

are all integers, then:

$$
2\pi
=
\frac{N_{\mathrm{cyc}}}
{m_*N_{1/2}}.
$$

This cannot hold exactly for finite integers because \(\pi\) is irrational.

Therefore at least one of the following must be true:

1. the transfer phase is not a finite root-of-unity cycle phase;
2. \(N_{1/2}\) is not an exact integer count in the continuum readout;
3. \(\tau_{1/2}=1/\omega_{\ell,I}\) is not the transfer half-period relation;
4. the first strain-coupled pole is not the same as the radar link frequency.

This is a real obstruction, not a detail to ignore.

Status:

`integer_cycle_step_obstruction`

### 65.5. Accepted non-circular options

Allowed ways forward:

1. spectral phase is generated by a continuous primitive action angle, not finite root-of-unity closure;
2. step count is a coarse-grained rate, not an exact integer per half-radar interval;
3. \(\omega_{\ell,I}\) is a response pole while \(\tau_{1/2}^{-1}\) is a derived radar scale related by a dimensionless invariant;
4. a richer grammar produces an irrational rotation number as a fixed point.

Each option must be declared before using gravity data.

`non_circular_step_options`

### 65.6. No-fit rule

Forbidden:

1. choose \(N_{1/2}\) to approximate \(1/\theta_*\) after comparing with \(G_N\);
2. hide irrationality by rounding \(2\pi\);
3. treat exact finite cycles and exact radar step counts as compatible when they are not;
4. use this rule to define \(\omega_{\ell,I}\) from \(G_N\).

Allowed:

1. use the obstruction to reject finite-cycle closure;
2. move to irrational rotation/fixed-point grammar;
3. treat step time as a continuum readout rate if derived independently.

`step_clock_no_fit_rule`

### 65.7. What is closed

Closed:

$$
\Delta\tau_{\mathrm{step}}
=
\frac{\Delta\tau_C}{N_{\mathrm{step}}}
$$

as a clock-readout route.

Closed:

finite root-of-unity cycle phase plus exact integer half-radar step count is not exactly compatible.

Open:

1. whether primitive grammar uses irrational rotation;
2. whether step count is only coarse-grained;
3. exact relation between response pole and radar link frequency;
4. numerical \(\omega_{\ell,I}\).
