## 177.9. Research Refinement: Distinguishability-Typed Facticization

The next pass found a sharper route than "add S2" or "add no ternary residue".
Those formulations are dangerous because they can simply rename the Born/Sorkin
wall.

The lower observation is about typing:

```text
D is a stable distinguishability relation.
R must be a facticization/readout witness relation for D.
```

If `R` is allowed to witness arbitrary whole-context facts, then a primitive
ternary residue can survive:

```text
context has singleton witnesses;
context has pairwise witnesses;
context also has one stable ternary R-witness;
all witnesses are finite, stable, and accounted.
```

That countermodel already killed the earlier hope:

```text
D_cl + F + Q + L => S2
```

So finite stability, orientation transport, and loss accounting are not enough.
The missing discipline is that `R` must be typed over `D`, not over arbitrary
context-level predicates.

Candidate principle:

```text
distinguishability_typed_constructive_facticization
```

Working statement:

```text
An admissible physical readout record is a finite, repeatable, refinement-stable
witness of a stable distinguishability relation D.  It is not a primitive
witness of an arbitrary whole-context predicate.  If a context-level invariant
is stable and physically readable, it must be generated by endpoint/proper-
subcontext D-witnesses or be declared as explicit loss/accounting outside the
readout sector.
```

Why this is different from stipulating Born:

```text
It does not say probabilities are |amplitude|^2.
It does not introduce Hilbert space.
It does not introduce a unitary group.
It does not ban all higher-context mathematics.
It only says that physical facticization records witness stable distinctions,
and stable distinctions are typed as relations, not primitive n-ary residues.
```

Immediate consequences if the principle holds:

```text
1. irreducible ternary readout residue is not typeable as an R-witness;
2. endpoint/proper-subcontext exhaustion becomes a typing consequence;
3. binary complement readout is natural because a D-witness has two oriented
   endpoints;
4. external randomization of repeatable R-records mixes frequencies affinely by
   finite record counting, not by an extra probability primitive;
5. nonlinear selectors can be rejected as violations of refinement/frequency
   stability rather than as "not Born";
6. hidden joint-only composite invariants are rejected when they have no
   product/refinement D-witness route;
7. Hilbert pressure comes from representing transported D-witness structure,
   not from postulating Hilbert space.
```

The Born-side proof route would then be:

```text
D-typed R
+ endpoint stability
+ repeatable finite record counting
+ affine mixture under external randomization
+ normalized oriented overlap transported by I
=> p = (1 + signed_overlap) / 2
```

and the phase-bundle projection gives:

```text
signed_overlap = 2 |a|^2 - 1
=> p = |a|^2
```

The still-open part is not the final algebra. The open part is deriving the
typed source:

```text
R witnesses D;
I transports D-witnesses;
finite refinement/product composition preserves R-witness admissibility;
no stable composite fact is admitted without a generated D-witness route.
```

This reframes the live wall:

```text
Old wall:
  derive Born or S2.

Better wall:
  derive distinguishability-typed constructive facticization from the
  context-first primitive base, or accept it as the next successor-base
  semantic law.
```

The decisive countermodel test is now:

```text
Can a finite theory satisfy C,O,I,R,D plus D-typed R, refinement stability,
operational equivalence, and loss accounting while still carrying an
irreducible stable ternary readout residue?
```

If yes, this route fails. If no, the Born/S2 wall is probably below probability
and inside the typing of facticization itself.

6. Exhaustion with explicit loss:
   every stable physical distinction is generated by finite R-routes or is
