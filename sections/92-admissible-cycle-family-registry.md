## 92. Admissible Cycle Family Registry

This section defines which cycles may enter the relative phase-cost closure test.

Status:

`admissible_cycle_family_registry_initialized`

### 92.1. Cycle family

A cycle family is a pre-declared set:

$$
\mathcal C_*
=
\{\gamma_a\}_{a\in A}
$$

generated from a grammar:

$$
\mathcal G_{\mathrm{cycle}}.
$$

The family must be fixed before evaluating:

$$
\lambda_{\theta C}^{(\gamma)}
=
\Theta_\gamma/\bar C_\gamma^K.
$$

Status:

`cycle_family_predeclared`

### 92.2. Admissibility conditions

A cycle:

$$
\gamma
$$

is admissible only if:

1. \(U_\gamma\) is derived, not fitted;
2. branch lift \(\Theta_\gamma\) is derived or pre-registered;
3. \(\bar C_\gamma^K\) is derived;
4. \(\bar C_\gamma^K>0\);
5. \(\Theta_\gamma\ne0\);
6. the cycle is not a duplicate power of another listed cycle unless powers are explicitly part of the test;
7. the cycle was not selected because it improves the residual.

Status:

`cycle_admissibility_conditions_defined`

### 92.3. Independence condition

Cycles must be independent in the declared cycle basis.

If:

$$
\gamma_c
=
\gamma_a\circ\gamma_b,
$$

then:

$$
\Theta_{\gamma_c}
=
\Theta_{\gamma_a}+\Theta_{\gamma_b}
$$

and:

$$
\bar C_{\gamma_c}^K
=
\bar C_{\gamma_a}^K+\bar C_{\gamma_b}^K
$$

must be checked as composition consistency, not counted as a new independent support for universality.

Status:

`cycle_independence_condition_defined`

### 92.4. Family classes

The theory recognizes three initial family classes.

Small contractible cycles:

$$
\mathcal C_{\mathrm{small}}
$$

use the principal continuity branch.

Topological cycles:

$$
\mathcal C_{\mathrm{top}}
$$

require derived winding data.

Clock-strain cycles:

$$
\mathcal C_{\chi}
$$

are generated by repeated closed clock-network strain updates and must be independent of gravitational calibration.

`cycle_family_classes_registered`

### 92.5. Closure ledger

For each cycle the ledger must contain:

| Field | Meaning |
|---|---|
| \(\gamma\) | cycle word |
| class | small / topological / clock-strain / other |
| \(U_\gamma\) | derived holonomy |
| \(\Theta_\gamma\) | branch-lifted phase |
| branch source | principal / winding / grammar |
| \(\bar C_\gamma^K\) | kernel-strain cycle cost |
| admissibility | pass/fail |
| residual role | calibration / validation / excluded-with-reason |

### 92.6. Calibration / validation split

If the theory uses some cycles to estimate:

$$
\lambda_{\theta C,I},
$$

then those cycles are calibration cycles.

A disjoint set must be reserved for validation:

$$
\mathcal C_{\mathrm{val}}
\cap
\mathcal C_{\mathrm{cal}}
=
\varnothing.
$$

Prediction requires validation cycles that were not used to choose:

1. branch rules;
2. admissible family;
3. scalar cost projection;
4. residual tolerance.

Status:

`cycle_calibration_validation_split_required`

### 92.7. No-fit rule

Forbidden:

1. remove cycles because they break universality;
2. add powers of a successful cycle as fake independent evidence;
3. move a failed validation cycle into excluded status after seeing the residual;
4. change family class after comparing with known \(\hbar\).

Allowed:

1. exclude cycles before testing because a required field is missing;
2. report family-splitting residuals;
3. reject the kernel-strain route for a family;
4. keep \(\lambda_{\theta C,I}\) provisional until validation passes.

Status:

`cycle_family_no_fit_rule`
